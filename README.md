# 🌿 Two Leaves Tea – Responsive Website Clone

A fully responsive front-end clone of the **Two Leaves Tea** website built using **HTML5** and **CSS3**. This project recreates the elegant and modern design of the original website, including its navigation bar, hero section, product showcase, typography, and interactive UI elements.

The goal of this project was to strengthen my understanding of responsive web design, CSS layouts, positioning techniques, custom fonts, and GitHub deployment while replicating a real-world website.
